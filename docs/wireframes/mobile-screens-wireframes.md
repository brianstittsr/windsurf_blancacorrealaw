These mobile wireframes implement the "Pathways to Belonging" visual theme while optimizing for smaller screens. Key design considerations include:

1. **Touch-Friendly Elements:** Larger touch targets for buttons and form elements
2. **Progressive Disclosure:** Information revealed gradually to avoid overwhelming small screens
3. **Simplified Navigation:** Clear back buttons and menu access
4. **One-Tap Actions:** Critical functions like "Call Now" for urgent help
5. **Vertical Organization:** Content stacked for easy scrolling rather than requiring horizontal navigation
6. **Visual Hierarchy:** Clear headings and concise content for quick scanning
7. **Consistent Branding:** Maintaining the warm color palette and journey motifs across all screens
